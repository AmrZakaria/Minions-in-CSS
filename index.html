Minions-in-CSS

Minions in Pure CSS

I had some few free time, so I decided to draw the "Minions" character using only HTML/CSS .. just to practice and have fun ..

I divided every character into separate elements .. ex: Hair, Smile, Clothes, Teeth, Shoes ... etc, using Pseudo-elements to handle this huge number of details ..

Hope you enjoy the Animation ..
